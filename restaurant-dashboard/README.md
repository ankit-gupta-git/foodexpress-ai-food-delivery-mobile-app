# FoodExpress: Restaurant Management Dashboard

This is the administrative interface for the FoodExpress delivery system. Built with **Next.js** and **Tailwind CSS**, it allows restaurant owners to manage their operations efficiently.

## ✨ Features

- **Orders Management**: Real-time monitoring of incoming orders.
- **Menu Management**: Dynamic control over dishes, prices, and availability.
- **Live Notifications**: Instant updates on order status changes.
- **Responsive Design**: Accessible from desktop and mobile browsers.

## 🛠️ Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Backend integration**: Firebase SDK

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the dashboard.
